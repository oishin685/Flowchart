# 取扱説明書（はじめて使う方向け）

この説明書は、**コードを書かずに使いたい人**向けです。

## 1. できること

- フローチャートを画面で作れる
- 条件分岐（if/else のような分岐）を設定できる
- 保存・読み込みができる
- JSON/Markdown で書き出せる
- 入力値を入れてシミュレーションできる

---

## 2. 起動手順

1. ターミナルを開く
2. プロジェクトフォルダに移動
3. 起動コマンドを実行

```bash
cd ~/Flowchart
npm install
npm run dev
```

4. ターミナルに出た `Local:` のURLをブラウザで開く
   - 例: `http://localhost:5173/`

---

## 3. 画面の見方

- **上**: 操作バー（保存・読込・Export・検証・シミュレーション）
- **左**: ノード追加
- **中央**: フローチャート編集キャンバス
- **右**: 選択中ノード/エッジのプロパティ編集
- **下**: ログ（エラー・検証結果・シミュレーション結果）

---

## 4. 使い方（最短）

1. 左でノードを追加（開始・条件・ステップ・終了）
2. 中央でノード同士を線でつなぐ
3. 右で選んだノードの内容を編集
4. 上の「検証」でエラー確認
5. 上の「シミュレーション」で動作確認

---

## 5. 保存と読込

- **保存**: 上部バーの「保存」
- **読込**: 上部バーの「読込」
- ブラウザの `localStorage` に保存されます

---

## 6. Export（外部共有・AI連携）

上部の `Export` から次が使えます。

- Download JSON
- Download Markdown
- Copy JSON
- Copy Markdown
- Generate AI Prompt

---

## 7. うまく表示されないとき

1. `npm run dev` を実行したターミナルを閉じない
2. `Local:` に表示されたURLを正確に開く
3. ブラウザを強制再読み込み（`Cmd + Shift + R`）
4. まだダメなら一度止めて再起動

```bash
Ctrl + C
npm run dev
```

---

## 8. よく使うキー

- `Delete`: 選択中のノード/エッジ削除
- `Ctrl + S` / `Cmd + S`: 保存
